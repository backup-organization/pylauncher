pyLauncher 10.5 and older requires Java 6

Java Installation Tutorial:
Step 1: Download and install Java 6: [CLICK HERE]
Step 2: Add Java to path: Open Control Panel;Open System and Security;
Open System;Click on Advanced System Settings;Click on Environment Variables;
Add to System Variables:
New variables:
Variable name:JAVA_HOME
Variable Value:"C:\Program Files (x86)\Java\jre6"
Variable to be edited:
Variable name:Path
Add this text to variable: ;%JAVA_HOME%\bin
Java 6 installed and added to Path; you can use pyLauncher 10.5 and older!
